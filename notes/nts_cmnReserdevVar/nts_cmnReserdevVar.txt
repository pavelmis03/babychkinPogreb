
RESERVED VALS

obj_name - переменная для загрузки объектов


COMMON VALS

action - действие, которое должен совершить объект по выполнении какого-то условия
draw_txt - рисуемый объектом текст
my_name - имя объекта 
type - тип объекта 
pre_rm - предыдущая комната
sound - запрос на проигрывание звука, звук
val - значение
iid - id объекта, который назначил действие текущему
init - флаг инициализации
time - время чего-либо
draw_spr - спрайт, который надо нарисовать
hp - жизни
maxHp - максимальные жизни
